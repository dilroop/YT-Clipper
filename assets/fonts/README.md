# Custom Emoji Fonts

If you want to use a specific emoji library (like Samsung Emojis), place your `.ttf` or `.ttc` file here.

### File Name Requirement
The Workflow 2 renderer specifically looks for:
`custom_emoji.ttf`

If you have a different name, either rename it to `custom_emoji.ttf` or update the path in `backend/videoprocessor/workflow2.py`.
